# BreezeUI
## BreezeUI is a light weight, open source and easy to use component building framework. Build UI  at a blazing fast speed.

### Stack Used
- HTML
- CSS


## Installation
Copy and paste the below link to the head of your html file and you are good to go!
```
<link rel="stylesheet" href="https://breez-u-i.netlify.app/style.css" />
```
